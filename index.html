Loan Default Risk — Logistic Regression

Built a baseline Machine Learning model to estimate the probability that a small business loan will default. The case uses a historical SBA dataset (cleaned & standardized), with a focus on explainability and fair evaluation. The study walks through data prep, feature engineering, train/test split, class imbalance handling, model fitting, and business oriented interpretation of coefficients.

  • Methods: logistic regression, regularization, class imbalance handling (class weights/threshold tuning), k-fold cross-validation, ROC-AUC/PR, confusion matrix.
  • Focus: interpretable coefficients & odds ratios, approval policy implications, and risk cutoffs.
  • Outcome: clear baseline for default risk scoring; shows how to set a threshold to balance approval rate vs. expected defaults for different risk appetites.

Deliverables: Notebook (ipynb) · Case Brief · Dataset (CSV)
